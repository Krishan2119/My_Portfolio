Email spam detection

Built a full-stack web app using the MERN stack to detect spam emails. Users can input email text, which is analyzed by a backend ML model (trained with NLP techniques) to classify it as spam or not. The app features a React.js UI, a Node.js/Express API, and MongoDB for data storage. This project demonstrates the integration of machine learning with modern web technologies.
